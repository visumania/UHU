The project consists of the IT management of a mobile phone company. Which can have two types of clients: Flat rate clients and mobile rate clients.

The project consists of the following files:
-Interface process
-Date Class
-Client Class
    -Client Flat Rate Class
    -Client Mobile Class
-Company
-Test1
-Test2
-Main

Test1 and Test2 are irrelevant files, they are only for testing the Date Class and the Client Classes.

The main program is in the Main file
